# Open Spot Notifier

Open Spot Notifier is a python script that helps ASU students get notifications the moment a spot opens up in their preferred class. The script uses Selenium and BeautifulSoup to load the ASU class search page, Discord.py to send real-time alerts, and incorporates proxy and header randomization for user anonymity.

## Features

- Real-time monitoring of class availability using Selenium and BeautifulSoup
- Immediate notifications via Discord using discord.py
- Proxy and header randomization for enhanced anonymity
- Configurable class and section preferences for personalized alerts
- Lightweight, cross-platform compatibility with minimal setup
